SPI_TFT_stm32f4

一、项目介绍

基于 STM32F4 单片机的 2.0 寸 SPI 接口 TFT 屏幕驱动，仅输入版本。适用于嵌入式显示实验、小项目开发使用。

二、功能支持

1、SPI 通信驱动
2、屏幕初始化
3、矩形色块绘制
4、字符显示

三、使用环境

1、单片机：STM32F4
2、开发环境：Keil MDK
3、屏幕驱动芯片：ST7789

四、文件结构

1、TFT.c/TFT.h：屏幕驱动代码
2、font.h：字库文件
3、delay.c/delay.h：延时代码
4、其他为 STM32 标准工程文件

五、硬件连接

1、BL（背光）：PA1 (高电平开启)
2、CS（片选）：，PA4 (低电平选中)
3、DC（参数识别）：PA3 (高电平数据、低电平命令码)
4、RST（复位）：PA2 (低电平复位、高电平工作)
5、SDA(主出从入):PA7
6、SCL（SPI时钟）:PA5
7、VCC：3.3v
8、GND

六、使用方法

1、将工程下载到本地
2、使用 Keil 打开工程文件
3、根据自己的硬件修改引脚配置
4、编译下载到开发板运行

七、注意事项

1、初始化之前花屏
2、清屏尺寸根据实际大小修改
3、接线错误可能导致屏幕不显示
4、RGB码与颜色数据格式错误对应可能导致颜色错误、色块填充大小不对
5、命令发送函数支持可变参数，调用时第二项输入传参数量
6、本项目仅为学习使用
